Tem interesse na área de desenvolvimento Web ?

Então se você deseja aprender a programar separei os primeiros passos, para você seguir e não tropeçar no meio do caminho.

Como nem todos sabem a programação especificamente para quem tem interesse em aprender a programar para Web é iniciada pelas linguagens de marcação HTML & CSS e só depois para linguagens de programação, JAVASCRIPT ETC...

HTML

HTML é o corpo Web. Onde fica escritas paragrafos titulos. é a o começo onde começamos a trabalhar com estrutura web, tags, semantica, acessibilidade é onde fica armazenado todo tipo de escrita visivel no site.

CSS

Diferente do HTML o CSS serve para estilizar o corpo(HTML), definir cor de fundo imagens de fundo, animações e efeitos. Ou seja é o que define qual o estilo da fonte será usado, qual cor que sera usada em qual parte do texto.

JS

JAVASCRIPT é ultilizado para captar interações do usúario na web. Após capitar é posasivel. Definir timer, adicionar novo estilo ao clique do usuario, pegar informações de outros sites por meio de (API) e colocar no atual.
