Introdução

1-1 Conhecendo a área

Antes de tudo é importante que você seja uma pessoa 100% dedicada e tenha persistencia e foco, na programação é comun encontramos prblemas que são resolvido com tempo e não em algumas horas mas sim alguns dias.

1-2 Quebrando Tabus

A programação é uma area saturada ?

A programação como todas as profisões tem pessoas cujo o objetivo dentro da area é somente o dinheiro. O mercado está saturado para esse tipo “Profissionais” que só buscam dinheiro. O que te trás um diferencial não é somente buscar o dinheiro mas fazer projetos pessoais que solucionam problemas e desenvolvover as suas habilidades com prática não somente na teoria.

É difícil de aprender ?

Sim mas, não impossivel. É importante entender isso desde o inicio, é relamente muito dificil não é atoa que existem poucos profissionais de verdade e capacitados, o que vai te tornar um é o foco e persistencia.

Depois que terminar de aprender HTML, CSS & JAVASCRIPT estou pronto ?

Não, você nunca estará pronto, como todos sabem a tecnologia avança constantemente. o que você precisa não é estar pronto mas sim preparado.

Como sei que estou preparado ?

A prática leva a perfeição. Estudo 1x e pratica o 3x mais. o que te deixa preparado é saber resolver problemas reais e ter praticado soluções para problemas reais.

Posso pesquisar respostas no google ?

Pode. Caso você tenha tentado resolver antes e não conseguiu. Mas não copie e cole. Antes leia o código linha por linha e procure entender cada detalhe que ele faz, não seja uma maquina de copiar e colar.

Conhecendo HTML

1-2 O que são tags ?

Para criar um site não podemos simplismente escrever o texto solto, temos que indentificar o que é cada parte. Texto, imagem, video, e assim o html por si só já aciona sentido ao texto ou midia adicionado e posiciona o conteudo em seu lugar especifico.

Exemplos de tags

<h1> Isto é uma tag de titulo </h1> <p> Isto é uma tag de parágrafo </p>

É importante notar que a maioria das tags html tem que ser fechada ex: A tag é aberta com "<” e “>” e fechada com sinial de “<” ”/” ”>”

Ou seja o que a gente que colocar dentro simplismente escrevemos a tag e adicionamos o texto ou midia dentro da Tag assim:

<html> <head> <tittle> Golfinhos </tittle> </head> <body> <h1> Golfinhos são legais </h1> <p> Você sabia que 90 % dos golfinhos estudam na six code ? </p> </body> </html>

Conhecendo HTML

1-3 Semântica

Semantica é o sentido do conteudo nesse caso a semantica é usada para dar sentido para o conteudo html de modo geral.

Exemplo: quando queremos indicar um titulo usamos a tag <h1> </h1> , já quando queremos um parágrafo usamos a tag <p> , mas e se quisermos coolcar uma imagem ? é usada a tag <img> e se quisermos um botao ? usamos a tag <button> </button> .

Qual é adifrença de usar a tag correta para o elemento correto ?

Maneira Correta:

<h1> A busca pelos Golfinhos </h1> <p> Para iniciarmos a busca é preciso entender o contexto, clique no botão para saber mais </p> <button> SAIBA MAIS </button>
